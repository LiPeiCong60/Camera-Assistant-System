## 当前文档口径

- 设备联动默认手机推流链路是 WebSocket NV21 上行 + JPEG 预览下行。
- WebRTC 代码和接口保留，但不是当前默认入口。
- 设备端抓拍保存在 `device_runtime/captures`，不自动同步后端历史。
- 手势抓拍有 3 秒倒计时，并通过设备端 overlay 和手机端 HUD 提示。
- 后端 AI 和设备端本地 AI 是两条不同链路。
- 树莓派现场优先使用 `DEVICE_RPI_PROFILE=performance`。

## 后续维护原则

- 改接口时同步更新 [接口契约](./接口契约.md)。
- 改部署变量时同步更新 [部署说明](./部署说明.md)。
- 改设备联动链路时同步更新 [技术架构与运行机制详解](./技术架构与运行机制详解.md) 和各 README。
- 不在文档中写入真实 API key、数据库密码、个人账号或用户照片路径。
