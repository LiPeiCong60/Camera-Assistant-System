# 文档清理说明

本轮按当前代码重新整理文档，目标是让文档反映真实实现，特别是设备联动已经改为 WebRTC 主链路，同时保留 WebSocket/JPEG fallback 的事实。

## 清理原则

- 稳定入口保留：根 README、模块 README、`docs` 下的主题文档。
- 阶段性设计稿、旧结论、重复说明应合并到稳定文档。
- 涉及删除文件时需要人工确认，避免误删仍有用的演示材料或历史资料。
