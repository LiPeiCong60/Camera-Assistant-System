# 接口契约

本文列出当前代码中的主要 HTTP/WebSocket 接口。

## 通用约定

后端和设备端大多数 HTTP 接口返回 JSON envelope：

```json
{
  "success": true,
  "message": "ok",
  "data": {}
}
```

地址约定：

- 业务后端：`http://host:8000/api`
- 设备运行时：`http://host:8001`
- 管理后台：`http://host:5173`

真机联调必须使用局域网 IP。

| `POST` | `/captures/upload` | 创建抓拍记录 |
| `POST` | `/captures/file` | 上传图片文件 |
| `POST` | `/ai/analyze-photo` | 创建照片分析任务 |
| `POST` | `/ai/analyze-background` | 创建背景分析任务 |

## 设备运行时接口

Base URL：`http://host:8001`

### 状态和配置

| 方法 | 路径 | 说明 |
| --- | --- | --- |
| `GET` | `/api/device/health` | 运行时健康检查 |
| `GET` | `/api/device/status` | 当前 session、云台、检测、手势、AI、overlay 状态 |
| `PATCH` | `/api/device/config` | 更新 overlay、检测、手势配置 |
| `GET` | `/api/device/preview.jpg` | 单张 JPEG 预览 |
| `WS` | `/api/device/preview-ws` | JPEG 预览流 |

`status.gesture_status.capture_countdown` 会返回手势抓拍倒计时：

```json
{
  "active": true,
  "remaining_s": 2.4,
  "duration_s": 3.0,
  "event": "open_fist_capture",
  "reason": "gesture_open_fist"
}
```

### 会话

| 方法 | 路径 | 说明 |
| --- | --- | --- |
| `POST` | `/api/device/session/open` | 打开设备会话 |
| `POST` | `/api/device/session/close` | 关闭设备会话 |

打开手机推流会话示例：

```json
{
  "session_code": "mobile-session",
  "stream_url": "mobile_push",
  "mirror_view": true,
  "start_mode": "MANUAL"
}
```

### 视频流
