# 开发进展记录

## 当前总览

- 主体模块已齐，项目已进入收口维护阶段
- 树莓派舵机路线已统一为“TTL 总线舵机 + 独立电源”

## 当前状态说明

当前项目已经完成：

- `backend`：FastAPI、PostgreSQL、用户/套餐/拍摄/AI 任务/AI Provider 配置/管理端接口
- `device_runtime`：本地 API、手动控制、模式切换、follow-mode、模板管理、AI apply-angle / apply-lock、抓拍触发
- `mobile_client`：登录、拍照、模板、上传、单图分析、背景分析、历史、设备联动
- `admin_web`：登录、工作台、用户管理、套餐管理、设备列表、拍摄记录、AI 任务、AI Provider 配置

当前重点已经转入：

- 树莓派真实联动补齐
- 文档整理与去除过时阶段稿
- 真机验收与部署准备
